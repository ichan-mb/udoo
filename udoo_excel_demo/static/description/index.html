This module provide some example use case for Dynamic Excel Engine

Highlights

⤳ Example 1
- Export/Import Excel on existing document
Form action menu: Go to any Sales/Purchase Order and use Export Excel or Import Excel in action menu.
List action menu: Select any Sales Order from list and use Export Excel in action menu.

⤳ Example 2
- Printout Excel on existing document, using report action
Form action menu: Go to any Sales Order and click print "Quotation / Order (.xlsx)".

⤳ Example 3
- Import Excel Files
Menu: Go to menu Sales > Reporting > Sample Import Sales Order

⤳ Example 4
- Create Excel Report
Menu: Go to menu Sales > Reporting > Sample Sales Report

⤳ Example 5
- Run Partner List Report, using report action
Menu: Go to menu Sales > Reporting > Sample Lead Report

⤳ Example 6
- Run Partner List Report, using report action
Menu: Go to menu Sales > Reporting > Partner List Report
