The Most Powerful Features

Unlike Odoo's base multi-website feature, our app allows you to list your products on multiple websites simultaneously. No longer limited to selecting just one or none, you can now showcase your products across different online platforms. Easily manage and expand your online presence by adding products and e-commerce categories to several websites at once.

Multi Website Product
  • Select a single website to a product
  • Select 2 or more websites to a product
  • Select all websites to a product (leave empty)
  • Bulk website assign for multiple products
  • Change product global publishing status
Multi Website Category Product
  • Select a single website to a category
  • Select 2 or more websites to a category
  • Select all websites to a category (leave empty)
  • Bulk website assign for multiple categories
Flexible And Intuitive Setup Interface
  • Access the wizard through multi-tree view selection
  • Access the wizard directly within form view
  • Change the target selection in action wizard
  • Assign all target selection to all website
