This module helps automatically captures and displays events by recording what activity was performed, who performed it, and when it occurred.
With this information, you can ensure that you're able to follow up and address any unexpected or undesirable events.

Highlights

There are several kinds of events that are logged:

⤳ Admin Activity

⤳ Data Access

To access the audit logs, on the homepage, go to Admin Settings > Technical > Audit > Logs. Within this tab, you can see a list of events that have been tracked under your Odoo's account.
