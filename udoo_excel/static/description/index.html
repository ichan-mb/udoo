This module helps developers manage Excel import, export and reporting in Odoo easily.

For demo, please install Dynamic Excel Engine Demo

Highlights

⤳ Effortless Setup
Build menus, actions, and views for Excel integration without rewrite from scratch.

⤳ Familiar Design
Create custom Excel templates using the standard Excel application.

⤳ Intuitive Data Mapping
Easily define data placement with simple instructions or the Odoo UI.

⤳ Automatic Generation
Odoo generates the final Excel file based on your instructions.

Dynamic Capabilities

Deeply Integration
Your module can seamlessly integrate with existing Odoo models, allowing users to select the desired model and export relevant data directly from within Odoo.
This simplifies the process of selecting data for export and ensures data consistency.

Customizable Templates
Allow to create and customize Excel templates according to their specific requirements, including formatting, layout, and formulas.

Dynamic Data Mapping
Provide a user-friendly interface to map Odoo model fields to corresponding columns in the Excel template.
This allows to define which data should be exported and where it should appear in the Excel file.

Data Transformation and Formatting
Enable to apply transformations and formatting to the exported data, such as converting dates to a specific format, adding calculations, or applying conditional formatting.

Multiple Sheet Support
Allow to export data into multiple sheets within a single Excel file, organizing the exported data based on different criteria or models.

Localization and Language Support
Ensure compatibility with different regional settings and languages, including support for localization-specific formats, currencies, and date formats.

Error Handling and Validation
Implement robust error handling and validation mechanisms to handle potential issues during the export process, such as missing or invalid data, ensuring data integrity in the exported Excel files.
