The Most Powerful Features

This module helps developers manage Excel import, export and reporting in Odoo easily.

For demo, please install Dynamic Excel Engine Demo

Deeply Integration
Enable seamless integration with existing Odoo models, facilitating effortless selection of the desired model and export of relevant data directly from within Odoo. This streamlined process simplifies data selection for export and ensures consistency in the exported data.
Dynamic Data Mapping
Offer an intuitive interface to map Odoo model fields with corresponding columns in the Excel template. This functionality empowers users to define the data to be exported and its placement within the Excel file.
Data Transformation and Formatting
Facilitate the application of transformations and formatting to the exported data, allowing for tasks such as converting dates to a specific format, adding calculations, or applying conditional formatting.
Customizable Templates
Facilitate the creation and customization of Excel templates based on specific requirements, encompassing various aspects such as formatting, layout, and formulas. This functionality empowers users to tailor the templates to their specific needs and preferences.
Multiple Sheet Support
Enable the export of data into multiple sheets within a single Excel file, providing the capability to organize the exported data based on various criteria or models.
Localization and Language Support
Ensure compatibility with different regional settings and languages, including support for localization-specific formats, currencies, and date formats.
Error Handling and Validation
Implement robust error handling and validation mechanisms to handle potential issues during the export process, such as missing or invalid data, ensuring data integrity in the exported Excel files.
